# Module Spec: The AI-Robot Brain (NVIDIA Isaac)

**Module**: 03
**Topic**: NVIDIA Isaac (Sim & ROS)
**Status**: Draft

## Objective
Teach AI-powered robotics using NVIDIA Isaac Sim and Isaac ROS.

## Learning Outcomes
- Understand Isaac ecosystem.
- Generate synthetic data in Isaac Sim.
- Implement perception pipelines.
- Use VSLAM and navigation.
- Deploy models to Jetson devices.

## Chapter Outline
1. **Introduction to NVIDIA Isaac**
2. **Isaac Sim Environment**
3. **Synthetic Data Generation**
4. **Isaac ROS Architecture**
5. **Perception Pipelines**
6. **VSLAM with Isaac**
7. **Navigation and Path Planning**
8. **Jetson Deployment Basics**

## Success Criteria
- Readers can run Isaac Sim scenes.
- Perception pipeline functions.
- Readers understand edge deployment.

## Constraints
- Requires NVIDIA GPU.

## Out of Scope
- Low-level Jetson driver coding.
